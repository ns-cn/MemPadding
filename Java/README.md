## 脚本使用说明

```shell
./Padding [type [thread-num]]
```
|     参数      | 说明                             |
|:-----------:|--------------------------------|
|    type     | 可选，内存填充类型（不填充选0、手动填充选1、自动填充选2） |
| thread-num  | 线程数量，可选，默认为10                  |

## 运行示例

### 不填充

以不填充的方式运行代码示例，并发线程数量为4

```shell
./Padding.sh 0 4
```

### 手动填充

以手动填充的的方式运行代码示例，并发线程数量为4

```shell
./Padding.sh 1 4
```

### 自动填充

以手动填充的的方式运行代码示例，并发线程数量为4

```shell
./Padding.sh 2 4
```
